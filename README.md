🚀 Habitify – Build Consistency, One Day at a Time

A modern, minimal habit tracking app designed to help you stay consistent, build streaks, and visualize your progress like GitHub contributions.

⸻

✨ Features

🔥 GitHub-style Heatmap
Visualize your daily consistency with a beautiful contribution grid

🏆 Streak System + Freeze 🧊
Maintain streaks and protect them using streak freeze logic

📊 Smart Weekly Analytics
Track your performance with dynamic charts

🔔 Smart Notifications
Get reminders based on your habit completion status

🎯 Daily Progress Tracking
See how much you’ve completed today

🎨 Clean & Premium UI
Minimal dark theme with smooth animations

🧠 How It Works
	•	Add habits from the library
	•	Complete them daily
	•	Maintain streaks
	•	Track consistency via heatmap
	•	Stay on track with smart reminders

⸻

🛠 Tech Stack
	•	💙 Flutter
	•	🧠 Provider (State Management)
	•	💾 SharedPreferences (Local Storage)
	•	📊 fl_chart (Analytics)
	•	🔔 flutter_local_notifications


  🧪 Key Highlights
	•	Custom GitHub-style heatmap implementation
	•	Smart notification logic (context-aware reminders)
	•	Streak + freeze system (advanced logic)
	•	Smooth UI transitions and animations
